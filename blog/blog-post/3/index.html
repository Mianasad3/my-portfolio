FrontEnd Technologies

Date: 20th March 2022

HTML

HTML stands for hypertext markup language, which is used to build websites and its content. The HTML tag specifies the beginning and end of an HTML element in an HTML document. The HTML Elements refers to the content within the tags, from start to end.

There are two types of Elements; block and inline. Block element contains a large block of content and always starts with a new line, such as <p> and <div> elements. On the other hand, inline does not start with no line and takes up less space. To provide additional info for a specific element, we use HTML attributes, with <a> tag and href attribute for the website link.

CSS

CSS stands for cascading style sheets. It is used to style documents written in HTML. A selector, a property, and a value are the parts of CSS rules. CSS selector gives command to the browser that which specific HTML element is selected that contains the CSS styling properties. Styling can be done on any HTML Element. Values are used to provide specific property to the element. The values can be any size assigned to font, any color to background or text. There are numerous kinds of values that can be assigned to a document.
